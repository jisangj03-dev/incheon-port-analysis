# 인천항 물류 데이터 분석

공공데이터를 기반으로 인천항의 컨테이너 물동량 흐름을 분석하는 정기 리서치 보고서 모음이다. 각 보고서는 **핵심 요약 → 분석 결과(차트·표) → 해석 → 한계·후속** 구조로, 데이터 출처와 정밀도 한계를 함께 밝힌다.

## 보고서 목록

- **[보고서 #01] 인천항 공컨테이너 물동량 분석 (2025)** — 연간 99만 TEU, 외국적 선사 88.2%, 설 연휴 계절성.
  → [reports/report_01_공컨테이너_물동량.md](reports/report_01_공컨테이너_물동량.md)
- **[보고서 #02] 인천항 공컨테이너 비율 분석 (2025)** — 전체 컨테이너의 약 28.8%가 빈 컨테이너, 수입 초과·재배치 부담 구조.
  → [reports/report_02_공컨테이너_비율.md](reports/report_02_공컨테이너_비율.md)

> #01이 "공컨 비중이 높다"는 관찰을 제시했다면, #02는 그것을 전체 물동량 대비 비율로 검증한다. 두 보고서는 이어지는 하나의 흐름이다.

## 사용 기술

Python, requests, pandas, matplotlib, 공공데이터 Open API (XML 파싱: `xml.etree.ElementTree`)

## 데이터 출처

- **공공데이터포털(data.go.kr)** — 인천항만공사 공컨테이너 화물 통계 API
- **인천지방해양수산청** — 「인천항 물동량」 월별 공표 통계 (전체 컨테이너 물동량)

## 저장소 구조

```
.
├── README.md          # 이 파일 — 보고서 허브
├── reports/           # 분석 보고서 (+ images/ 차트)
│   ├── report_01_공컨테이너_물동량.md
│   └── report_02_공컨테이너_비율.md
├── analysis/          # 재현용 코드·데이터 (config.py는 .gitignore로 제외)
└── docs/              # API 탐색·벤치마킹·TODO 기록
```

재현 방법은 각 보고서의 「재현 방법」 절을 참고한다(코드는 모두 `analysis/` 폴더에서 실행).
