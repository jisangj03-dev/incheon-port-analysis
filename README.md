# 인천항 물류 데이터 분석

공공데이터를 기반으로 인천항의 컨테이너 물동량 흐름을 분석하는 정기 리서치 보고서 모음이다. 각 보고서는 **핵심 요약 → 분석 결과(차트·표) → 해석 → 한계·후속** 구조로, 데이터 출처와 정밀도 한계를 함께 밝힌다.

## 보고서 목록

- **[보고서 #01] 인천항 공컨테이너 물동량 분석 (2025)** — 연간 99만 TEU, 외국적 선사 88.2%, 설 연휴 계절성.
  → [보고서 전문 보기 →](reports/report_01_공컨테이너_물동량.md)
- **[보고서 #02] 인천항 공컨테이너 비율 분석 (2025)** — 전체 컨테이너의 약 28.8%가 빈 컨테이너, 수입 초과·재배치 부담 구조.
  → [보고서 전문 보기 →](reports/report_02_공컨테이너_비율.md)
- **[보고서 #03] 인천항 공컨테이너 수출입 방향 분석 (2025)** — 빈 컨테이너의 85.1%가 수출 방향(수입의 6.1배), 불균형의 방향 규명.
  → [보고서 전문 보기 →](reports/report_03_공컨테이너_수출입방향.md)
- **[보고서 #04] 인천항 공컨테이너 불균형의 지속성 — 연도별 추세 (2022–2025)** — 48개월 전부 수출 방향 우위(월별 최소 3.9배), 불균형은 특정 연도가 아니라 구조다.
  → [보고서 전문 보기 →](reports/report_04_공컨테이너_연도별추세.md)
- **[보고서 #05] 인천항 컨테이너 수지 — 적컨은 수입, 공컨은 수출 (2022–2024)** — 전체는 방향 균형(1.0배)이지만 공컨을 빼면 적컨은 수입 우위(2배 안팎), 공컨의 수출 쏠림과 정반대 방향.
  → [보고서 전문 보기 →](reports/report_05_컨테이너_수지.md)

> #01이 "공컨 비중이 높다"는 관찰을 제시하고, #02가 전체 물동량 대비 28.8%라는 비율로 검증했으며, #03이 그 불균형의 방향(85.1% 수출 방향)을 규명했고, #04가 이 구조가 2022~2025년 48개월 내내 지속됨을 확인했으며, #05가 전체에서 공컨을 빼면 적컨이 수입으로 쏠린다는 거울상(공컨 수출 쏠림의 반대)을 확인했다 — 관찰→검증→규명→지속성→수지.

## 사용 기술

Python(requests·pandas·matplotlib), 공공데이터 Open API·XML 파싱, hwpx 공표자료 파싱(zipfile+XML), git·GitHub·GitHub Pages, 검증 게이트 설계(기발행 수치 회귀 대조·내적 지문 검산), Wayback Machine을 통한 사멸 API 문서 복원

## 검증 방식

- 모든 발행 수치는 저장소에 커밋된 수집 원시 CSV에서 재현 가능하다.
- 발행 전 검증 게이트를 통과한다: 기존 발행 수치와의 회귀 대조, 스키마·완결성 확인, 내적 검산(방향합=총합, 규격 환산식).
- 오류는 삭제하지 않고 정정 표기로 남긴다. 판정 기준은 결과 확인 전에 문서로 선커밋한다(docs/ 의 주제검증·정정 기록 참조).

## 데이터 출처

- **공공데이터포털(data.go.kr)** — 인천항만공사 공컨테이너 화물 통계 API
- **인천지방해양수산청** — 「인천항 물동량」 월별 공표 통계 (전체 컨테이너 물동량)
