# 인천항 컨테이너 물동량 분석 (공공데이터 API 활용)

공공데이터포털의 인천항만공사 API를 Python으로 호출해, 인천항의 월별 공컨테이너 물동량을 집계하고 시각화한 프로젝트입니다.

## 사용 데이터

- **공공데이터포털(data.go.kr) — 인천항만공사 공컨테이너 화물 통계정보 API**
- 응답 형식: XML
- 조회 대상: 2025년 1~12월 인천항 공컨테이너 반입·반출 실적(TEU)

## 분석 과정

1. **API 검토 및 선별**
   공공데이터포털에서 인천항 관련 API 4개를 후보로 놓고, "인천항 월별 수출입 컨테이너 물동량"이라는 목적에 맞는지 직접 호출·검증했습니다. 이 중 3개는 목적에 맞지 않아 제외했습니다.
   - *연안 물동량 통계 API* — 수출입·환적 구분 없이 **연안(국내) 물동량만** 제공
   - *전국 수출입 컨테이너 API(해양수산부)* — 월별·수출입 구분은 있으나 **국내 항만 구분이 없고 전국을 교역상대국별로 집계**해, 인천항만 따로 뽑을 수 없음
   - (그 외 후보 1개도 항목 구조가 목적과 불일치)
2. **API 선정** — 위 검증을 거쳐 조건에 맞는 **공컨테이너 통계 API**를 선정했습니다.
3. **API 호출** — Python `requests`로 2025년 1~12월 데이터를 한 번에 요청했습니다.
4. **파싱 및 집계** — 받은 XML을 파싱한 뒤 `pandas`로 월별 집계했습니다. 원본은 한 달에 반입·반출 구분과 터미널 구분이 나뉘어 **64행**이었고, 월(mm) 기준으로 합쳐 **12행**으로 정리했습니다.
5. **시각화** — `matplotlib`로 월별 추이 꺾은선 그래프를 그리고 PNG로 저장했습니다.

## 사용 기술

- Python
- requests (API 호출)
- pandas (데이터 집계)
- matplotlib (시각화)
- 공공데이터 Open API
- XML 파싱 (`xml.etree.ElementTree`)

## 주요 발견

- **외국적 공컨테이너가 전체의 약 88~90%**를 차지합니다.
- **2월이 물동량 최저**입니다. 설 연휴로 조업일수가 줄어든 영향으로 보입니다.
- 공컨테이너(빈 컨테이너) 비중이 높다는 것은 **무역 불균형과 빈 컨테이너 재배치 부담**을 보여주는 신호로 해석할 수 있습니다. 수출입 물량 차이가 클수록 빈 컨테이너를 다시 실어 나르는 비용이 커지기 때문입니다.

> 참고: 이번 분석은 **공컨테이너(빈 컨테이너)** 통계로, 인천항 전체 컨테이너 물동량의 한 단면입니다. 적(積)컨테이너를 포함한 전체 물동량 분석은 후속 과제로 남겨두었습니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `fetch_test.py` | API를 처음 호출해 원본 XML을 그대로 출력해 본 테스트 |
| `parse_test.py` | XML을 파싱해 pandas 표로 만들어 본 테스트 (여기서 연안 물동량만 제공됨을 확인) |
| `verify_new_api.py` | 후보 API(전국 수출입 컨테이너)가 목적에 맞는지 검증한 스크립트 |
| `gonggong_container.py` | 최종 선정한 공컨테이너 API를 호출해 원본 64행을 확인 |
| `analyze_container.py` | 64행을 월별 12행으로 집계하고 CSV로 저장 |
| `chart_container.py` | 월별 추이를 꺾은선 그래프(PNG)로 시각화 |
| `config_example.py` | 인증키 입력용 템플릿 (복사 후 `config.py`로 사용) |
| `container_2025.csv` | 월별 집계 결과 |
| `container_chart_2025.png` | 월별 추이 그래프 |
| `API_탐색결과.md` | API 탐색 과정에서 확인한 내용 정리 |

## 실행 방법

1. 공공데이터포털에서 인증키를 발급받습니다.
2. `config_example.py`를 복사해 `config.py`로 이름을 바꾸고, 발급받은 인증키를 넣습니다.
   (`config.py`는 `.gitignore`로 제외되어 저장소에 올라가지 않습니다 — 인증키 노출 방지)
3. 필요한 라이브러리를 설치합니다.
   ```
   pip install requests pandas matplotlib
   ```
4. 스크립트를 실행합니다.
   ```
   python analyze_container.py    # 월별 집계 + CSV 저장
   python chart_container.py      # 그래프 PNG 저장
   ```
