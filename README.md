# 인천항 물류 데이터 분석

공공데이터와 공식 통계를 바탕으로 인천항의 컨테이너 물동량 흐름을 하나씩 정량 분석하는 리서치 모음이다. 각 보고서는 **핵심 요약 → 분석 결과(차트·표) → 해석 → 한계·후속** 구조로, 데이터 출처와 정밀도 한계를 함께 밝힌다.

## 프로젝트 소개

- **주제**: 인천항 컨테이너 물동량의 구조적 특성(수출입 불균형, 빈 컨테이너 재배치 부담)을 공개 데이터로 실증
- **접근**: 목적에 맞는 데이터를 직접 탐색·검증하고, API가 없으면 공식 통계 원문으로 보완한다. **추정치를 만들지 않고 공표된 실측값만 사용**한다.
- **결과물**: 재현 가능한 Python 스크립트 + 월별 집계 데이터(CSV) + 시각화(PNG) + 해석 보고서(Markdown)

## 사용 기술

- **Python** — requests(API 호출), pandas(집계·비율 계산), matplotlib(시각화)
- **데이터 파싱** — `xml.etree.ElementTree`(공공 API XML), zipfile·정규식(hwpx 공표 원문)
- **데이터 출처**
  - 공공데이터포털(data.go.kr) — 인천항만공사 공컨테이너 화물 통계 API
  - 인천지방해양수산청 — 「인천항 물동량」 월별 공표 통계

## 재현 방법

모든 코드는 `analysis/` 폴더 안에서 실행한다(상대 경로 기준).

1. **인증키 발급** — 공공데이터포털(data.go.kr)에서 인천항만공사 공컨테이너 화물 통계 API 활용 신청 후 인증키를 받는다.
2. **인증키 설정** — `analysis/config_example.py`를 복사해 `analysis/config.py`로 이름을 바꾸고 발급받은 키를 넣는다.
   ```
   # analysis/config.py
   SERVICE_KEY = "여기에_본인의_공공데이터포털_인증키를_넣으세요"
   ```
   > `config.py`는 `.gitignore`로 제외되어 저장소에 올라가지 않는다 — 인증키 노출 방지 구조.
3. **라이브러리 설치**
   ```
   pip install requests pandas matplotlib
   ```
4. **실행**
   ```
   cd analysis
   python analyze_container.py    # #01: 공컨 월별 집계 + CSV
   python chart_container.py      # #01: 차트 PNG
   python calc_ratio.py           # #02: 비율 계산 + 차트 PNG
   ```
