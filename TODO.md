## 1. 깃허브에 올리기 (push)
아직 원격 저장소에 안 올림. 로컬 커밋만 있는 상태.
```
git remote add origin <깃허브_저장소_주소>
git push -u origin main
```
- `config.py`(인증키)는 `.gitignore`로 제외되어 자동으로 안 올라감 — 그대로 push하면 됨.

## 2. 전체 물동량 대비 공컨테이너 비율 분석 (핵심 후속 과제)
지금 데이터는 **공(빈)컨테이너**만. 인천항 **전체 컨테이너(적+공) 물동량** 데이터를 구해서
"전체 중 공컨이 몇 %인지"를 월별로 계산하면 무역 불균형 흐름을 더 잘 보여줄 수 있음.
- 할 일: 인천항 전체 컨테이너 월별 물동량을 제공하는 API/데이터 찾기 (공공데이터포털 재탐색)
- 찾으면: `전체 - 공컨 = 적컨`, `공컨/전체 = 공컨비율` 월별 표로 정리 → 차트

## 3. 시각화 보강 (여유 될 때)
