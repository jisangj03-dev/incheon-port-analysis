## 1. 프로젝트 개요

- **이름**: 「인천항 물동량 분석」 (역량부서 프로젝트)
- **목적**: **취업 포트폴리오** — 선광(인천항 컨테이너 터미널 운영사) 및 대기업 물류 직군 지원용
- **성격**: 공공데이터를 기반으로 인천항 컨테이너 물동량을 분석하는 **정기 리서치 보고서 사이트**
- **핵심 컨셉**: 데이터를 직접 탐색·검증하고, 추정치 없이 **공표된 실측값만** 사용해, 해석까지 담은 보고서를 번호를 붙여(#01, #02 …) 쌓아 나간다.

## 2. 완성된 보고서

| 번호 | 제목 | 핵심 결론 |
|------|------|-----------|
| **#01** | 인천항 공컨테이너 물동량 분석 (2025) | 연간 **99만 TEU**(991,170), 외국적 선사 **88.2%**, 2월 설 연휴 계절성(-31.2%) |
| **#02** | 인천항 공컨테이너 비율 분석 (2025) | 전체 컨테이너의 약 **28.8%가 빈 컨테이너** → #01의 "수출입 불균형·재배치 부담" 가설을 정량 검증 |

- #01 → #02는 하나의 흐름이다: #01이 "공컨 비중이 높다"는 관찰을 제시했고, #02가 이를 **전체 물동량 대비 비율**로 검증했다.
- #02의 핵심 난제는 **분모(인천항 월별 전체 컨테이너 TEU) 확보**였다. API 6종을 검증했으나 모두 부적합 → 인천지방해양수산청 공표 통계 원문(hwpx)에서 직접 추출해 해결했다. (상세: `docs/API_탐색결과.md`)

## 4. 환경 주의사항 ⚠️

- **작업 폴더**: `문서\역량부서\인천항물동량`
  - ⚠️ **`수출서류랩`(다른 프로젝트)과 절대 섞지 말 것.** 완전히 별개 프로젝트다.
- **클로드 코드를 켤 때 반드시 경로부터 확인**한다. 엉뚱한 폴더에서 작업하면 파일이 뒤섞인다.
- **`config.py`(공공데이터 인증키)** 는 `.gitignore`로 제외되어 저장소에 올라가지 않는다. 새로 받으면 `analysis/config_example.py`를 복사해 `analysis/config.py`로 만들고 키를 넣는다. **절대 커밋하지 말 것.**
- **push가 2분 타임아웃**나면, 커밋까지만 하고 멈춘 뒤 **별도 PowerShell 창에서 `git push origin main`** 을 직접 실행한다.
