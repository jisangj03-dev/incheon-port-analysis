## 0. 검증 경위와 사전 관측 고지

시간 순서: 노출 이력 ②(사고) → 검증 지위 강등 설계 → §4 기준 선언(채팅) → 프로브 #07-C1 실행. 즉 **§4의 선커밋 기준은 프로브 실행 이전에 선언되었다**(2026-07-22).

**노출 이력 ① — probe_02b 원시 저장(2026-07-13).** #04 주제검증기의 가용 기간 프로브가 2026년 원시 응답 30행(1~5월)을 `analysis/probe/probe_2026_raw.csv`로 저장했다. 존재 월 확인 목적이었고 집계·방향 라벨 결합은 실시하지 않았다. 파일은 감사추적 자산으로 저장소에 유지한다(삭제 금지).

**노출 이력 ② — 챗 컬럼 오인 노출 사고(2026-07-22, 이 세션).** 챗이 위 파일의 구조 점검(월·코드 행수만 볼 의도)에서 컬럼을 위치 기준으로, 그것도 2025 CSV 순서로 가정했다. 실제 파일은 원시 태그 순서(esbCntcDt 선두)였고, 그 결과 **esbCntcDt·forEmpTeu·forEmp_10·forEmp_20 값이 30행 전부, 방향(GInOut)·월(mm) 라벨 없이 챗에게 노출**되었다. 미노출: GInOut·mm·ocCt·forEmp_40·forEmp_99·korEmp 전 필드·yyyy·prtAtCode.
- 실질 판정(보수): 적재일 배치로 월 귀속이, 기발행 구조(#03·#04: 수출 ≈ 수입×6)로 큰 값의 방향이 사실상 추정 가능하다. → **2026-01~05의 방향·배율 축은 "부분 관측"으로 강등하고 검증(V) 지위에서 제외한다.** 해당 구간의 방향·배율은 분석 세션에서 "확정 집계"(참고 통계)로만 보고하고 본 §0을 별표로 인용한다.
- 40ft·규격 격차 축은 forEmp_40·korEmp 전 필드·방향 결합이 미노출이므로 미관측을 유지한다. 단 Teu 항등식을 경유한 간접 추정 가능성이 원리상 존재함을 고지한다(_99 미노출로 확정 불가).
- 교훈 재적중: "태그명(헤더명) 기준 파싱, 위치 금지"가 챗 자신에게 재적중했다.

**미관측 확인.** 2026-06은 probe_02b 시점(07-13) 미적재였고(적재일 2026-07-14), 노출 이력 ②의 범위(1~5월)에도 포함되지 않는다. → **완전 미관측. V1·V2의 검증 코어.**

## 1. 후보와 판정
- **부두별(신항/남항)** — 이월 유지. hwpx 4개년 미축적(2024 12개월만), 천TEU 반올림 정밀도 한계. 조건 불변: 4개년 축적 시 재검토.
- **방향별 계절성(#06 §5 후속)** — 유보 유지. 2022~2025 48개월 월별 방향 값이 #04·#06에서 전수 처리·관측되어 미관측 축이 얇다. 선커밋을 보전할 검증 영역이 성립하지 않는다.
- **2026년 표본외 연장 검증(+전년 동기 비교)** — **채택.** ① 2026-06 완전 미관측 + 규격 격차 축 미관측 → 선커밋 보전 가능. ② 기발행 구조 명제를 실패 가능한 신규 데이터에 노출하는 표본외 검증. ③ 단일 소스·기존 파이프라인, 신규 수집 수작업 0.

## 2. 채택 주제
- **질문:** 기발행 구조 명제 — #04 "공컨 수출 방향 우위(48개월 전부)"와 #06 "수입 40ft 쏠림 격차(42/48개월, 2022-10부터 39개월 연속)" — 가 발행 이후 적재된 2026년 상반기(1~6월)에서도 성립하는가. 부수: 전년 동기(2025-01~06) 대비 규모 비교(참고 통계).
- **데이터:** 공컨 API(ipaEmpConCargoInfo) 단일 소스, 2026-01~06. 분석 모집단 = ocCt=1(외항). 방향 = GInOut 1(수입)·2(수출), 환적(3·4)은 별도 표기.
- **정의:** 방향별 TEU = forEmpTeu + korEmpTeu. 규격 박스 = forEmp_x + korEmp_x. 40ft 비중 = _40합 ÷ (_10+_20+_40+_99 합), **박스 수 기준**.

## 4. 선커밋 판정 기준 (분석 세션은 이 기준으로만 판정한다)

### 검증 항목 (선언: 2026-07-22, 프로브 #07-C1 실행 이전)
- **V1 (방향 지속, 검증 — 2026-06 단독):** 공컨 수출 TEU > 수입 TEU (외항, G2 vs G1).
- **V2 (배율 하한, 검증 — 2026-06 단독):** 월 배율(수출 TEU ÷ 수입 TEU) ≥ 3.9. 근거 = #04 발행 48개월 월별 최소 3.9배(2023-12). 하회 시 국면 전환 신호로 판정.
- **V3 (규격 격차 부호, 검증 — S 전 6개월 월별):** (수입 40ft 박스 비중 − 수출 40ft 박스 비중) > 0. 6개월 전부 성립 시 PASS. 통과 시 연속 성립 기록이 2022-10부터 45개월(39+6)로 연장된다.

- **참고 통계(검증 아님):** ⓐ 2026-01~05 방향·배율 확정 집계 — §0 노출 이력 ② 별표 필수. ⓑ 반기(01~06) 합산 배율. ⓒ 전년 동기(2025-01~06, 기발행 데이터) 대비 방향별 TEU 증감.

1. **회귀 앵커 재현:** 2025 재수집 시 연간 소수값 991,170.0 / 843,837.75(수출) / 139,418.25(수입) / 7,914.0(환적계) 및 #06 확정 앵커(수입 40ft 91.1426%·66,463/72,922 / 수출 52.4408%·284,468/542,456) 불일치 → 정지(교훈 27).
2. **Teu 항등식:** 전 행 |Teu − (0.5×_10 + 1×_20 + 2×_40 + 2.25×_99)| > 0.01 → 정지.
3. **정수성:** 2026 규격 박스 필드 비정수 발견 → 정지·기록(4개년 전수 정수 [확정]과 대조).
4. **월 소실:** S(01~06) 중 결손 → 정지, 프로브 자체 의심(0패딩·키·네트워크) 후 재확인.

### 판정 규칙
- V1·V2는 06월 단독 판정(§0 강등의 귀결). V3는 월별 전수 공개 — 1개월이라도 미성립이면 V3 FAIL이되 성립 월 수를 그대로 기록한다.
- 분모 0 월 = 미성립 보수 처리. 저볼륨 월 제외 금지(#06 승계).
- 어느 V든 FAIL → 기준 사후수정 없이 이탈 기록, "구조 변화 신호"로 발행한다. FAIL도 발행 대상이다.

### 표기·방법 규칙
- 규격 비중은 박스 수로만(TEU 금지 — 순환). 비중·격차는 반올림 전 값으로 계산해 소수 1자리 표기(#06 각주 1 방식).
- 파싱은 태그명 기준, 위치 파싱 금지(§0 사고 실증). 정식 수집 CSV(container_2026_direction.csv)는 2025 형식 컬럼 순서로 저장.
- 2026 전체 = 잠정치 각주: 적재일(esbCntcDt) 병기 + 수집일 명기 + 사후 수정 가능성 고지.
- 인과·목적 서술 금지. _99 = "기타 규격" [미확인] 유지.
