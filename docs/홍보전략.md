# 홍보 전략 v1 (2026-07-14)

## 목표 정의
이 프로젝트의 홍보 목표는 오디언스·조회수가 아니라 **취업 신호**다. 성공 지표는 채용 과정(서류·면접)에서 이 저장소 링크가 실제로 열리고 프로젝트가 대화의 소재가 되는 것이다. 대중 채널(유튜브 등)은 범위 밖이다.

## 3단 구조
- 1단계 — 도착지 정비: README 30초 테스트. 처음 방문한 채용담당자가 30초 안에 "무엇을·어떻게·누가 직접"을 파악하게 한다. [2026-07-14 완료]
- 2단계 — 유통 경로 = 취업 서류: 이력서 프로젝트 단락 + 포트폴리오 1장. (코워크 첫 실사용 후보)
- 3단계 — 공개 채널 1개 = 링크드인: 프로필 정비 + 프로젝트 등록 + 시리즈 1 완결 포스트 1건.

## 관찰 게이트
3단계 게시 후 4주 반응 관찰(프로필 조회·링크 유입·연결/문의). 반응 데이터가 열 때만 확장을 검토한다(과정 기록 콘텐츠 등). 데이터가 게이트를 연다.

## 하지 않는 것
- 대중 채널 선행, 과정 기록 콘텐츠 선행(보류), 광고 의존 모델(기각 유지).
- AI 활용 은폐 — README·보고서 명시 유지(정직 하한선).
