- 3개년 합산: **수입 방향 40ft 84.0%**(총 157,592박스) vs **수출 방향 51.9%**(총 1,671,790박스). 연도별 격차 +7.5 / +33.0 / +37.8%p — 확대 추세.
- 40ft 편중의 시간 구조: 수입 방향이 2022 58.4% → 2023 84.7% → 2024 90.9%로 심화. 2022→2023 경계는 #04가 관측한 "2022 이례 vs 2023~ 새 균형" 경계와 일치(체인 교차 정합 — 단정 아닌 관측).
- `_10`: 수입·환적 방향 전 행 0. 수출 방향 3개년 합 4박스(2022년 1·2023년 3, 비중 0.0002%). ※ 세션 중 챗이 "_10 전 구간 0"으로 축약 발언 → 실행 출력으로 정정(전칭 0 아님).
- `_99`: 수입 0.10% / 수출 2.03% — 잔여 수준.
- **회귀 앵커 재현:** 프로브의 방향별 TEU 합에서 수출/수입 배율 = **27.32 / 6.03 / 7.70** — #04 발행값과 일치. 연간 공컨 합 873,917 / 999,991 / 1,044,970 — 발행값(873,918 / 999,990 / 1,044,970) 대비 ±1 TEU는 방향별 반올림 표시 후 합산한 표시 아티팩트. 분석 세션에서 원자료 소수 앵커(예: 2024 = 1,044,969.5)로 게이트 재수행.

## 4. 선커밋 판정 기준 (분석 세션은 이 기준으로만 판정한다)

### 검증 항목
- **V1 (연도, 2022~2024) — [관측 완료, 실재성 근거]:** 수입 방향 40ft 비중 ≥ 수출 방향 40ft 비중 + 5%p, 매 연도. 관측: +7.5 / +33.0 / +37.8%p 성립.
- **V1-2025 (연도, 미관측 — 선커밋):** 2025년에도 V1이 성립하는가. **실패 가능.**
- **V2 (월 단위, 미관측 — 핵심 선커밋):** 2022-01~2025-12 48개월 중 **39개월 이상(≥80%)** 에서 월별 (수입 40ft% − 수출 40ft%) > 0. **실패 가능.**

### 게이트 (정지 조건)
1. 규격식 검산: 외항 전 행에서 `0.5×_10 + 1×_20 + 2×_40 + 2.25×_99 = TEU` 성립(외국적·한국적 각각).
2. 기발행 소수 앵커 회귀: 연간 공컨(2025 = 991,170.0 / 2024 = 1,044,969.5 등)·배율(27.32 / 6.03 / 7.70 / 6.05) 재현.
3. 불일치 시 **전면 정지·원인 규명 우선**(계속 진행 금지).

### 판정 규칙
- V1-2025 ∧ V2 통과 → **완전 채택**(4개년 48개월).
- V1-2025 통과 ∧ V2 미달 → 연도 단위 명제로 **축소 채택** + 월 단위 이탈을 별도 절에 기록(기준 사후수정 금지).
- V1-2025 미달 → "2025 구조 변화 연도" 별도 서술 vs 3개년 축소 채택은 **회장 판정**(챗은 양쪽 초안만 제시). 어느 쪽이든 이탈 기록.
- 게이트 불일치 → 판정 자체를 하지 않는다.

### 저볼륨 규칙 (미리 못박는다)
- 월별 판정 표에 방향별 박스 수를 병기한다. 어느 방향이든 월 1,000박스 미만이면 저볼륨 각주.
- 분모 0인 월(해당 방향 박스 0)은 **미성립으로 센다**(보수 처리) + 각주.
- 저볼륨 월을 V2 판정에서 **제외하지 않는다** — 제외를 허용하는 순간 기준이 무너진다.

### 용어·표기 규칙
- "규격" = 컨테이너 사이즈 축(10/20/40 + 기타 `_99`). "박스" = 컨테이너 개수(자연 단위). TEU = 20ft 환산 단위.
- **규격 구성 비중은 박스 수로만 계산한다. TEU로 규격 비중을 재지 않는다** — TEU는 사이즈 가중(40ft=2TEU)이 내재되어 순환 논증이 된다. 물량 비교가 필요한 문장에만 TEU 병기.
- `_99` 정의 [미확인]: "기타 규격" 잔여 카테고리로만 서술. 45ft 등 특정 규격 단정 금지.
- 인과·목적 서술 금지: "선사가 ~하려고" 류 단정 금지. 발행 본문은 관측된 비대칭의 서술까지. 해석 가설은 §3(회장 직접)에서 가설임을 명시하고 제시.

2. 발행 표기: 비중은 소수 1자리 %, 박스 수는 정수(반올림 표기 명시). 게이트 앵커는 원자료 소수값.

## 부록 A — 프로브 요지 (챗 컨테이너, 2026-07-15)

집계 로직: 3개년 direction CSV → ocCt=1 필터 → (yyyy, GInOut)별로 `forEmp_{sz}+korEmp_{sz}` 박스 합산 및 `forEmpTeu+korEmpTeu` 합산 → 40ft 비중 = `_40 / (_10+_20+_40+_99)`.

```python
import csv
from collections import defaultdict
files = ['container_2022_direction.csv','container_2023_direction.csv','container_2024_direction.csv']
def num(x):
    try: return float(x)
    except: return 0.0
agg = defaultdict(lambda: defaultdict(float)); teu = defaultdict(float)
for fn in files:
    for r in csv.DictReader(open(fn, encoding='utf-8-sig')):
        if r.get('ocCt') != '1': continue
        k = (r['yyyy'], r['GInOut'])
        for sz in ['10','20','40','99']:
            agg[k][sz] += num(r.get(f'forEmp_{sz}')) + num(r.get(f'korEmp_{sz}'))
        teu[k] += num(r.get('forEmpTeu')) + num(r.get('korEmpTeu'))
for (y,g), d in sorted(agg.items()):
    tot = d['10']+d['20']+d['40']+d['99']
    print(y, g, {s:int(d[s]) for s in d}, f"40ft={100*d['40']/tot:.1f}%", f"TEU={teu[(y,g)]:,.0f}")
```

2025는 커버리지 확인만 실행(월·GInOut·컬럼 존재 확인, 규격 집계 미실행 — §0 고지).
