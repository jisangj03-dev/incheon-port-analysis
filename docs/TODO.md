아직 원격 저장소에 안 올림. 로컬 커밋만 있는 상태.
```
git remote add origin <깃허브_저장소_주소>
git push -u origin main
```
- `config.py`(인증키)는 `.gitignore`로 제외되어 자동으로 안 올라감 — 그대로 push하면 됨.
