앞으로 이 사이트를 발전시킬 때 참고할 조사 내용이다.

## 1. KMI 한국해양수산개발원 — 도메인 보고서 표준
- 국무총리 산하 해양수산 국책연구기관. 동향분석/주간 해운시장포커스/국제물류위클리/항만과 산업/인포그래픽 등 정기 보고서 시리즈 다수 운영.
- 훔칠 것:
  - 보고서 호수 넘버링 (제223호 식) → 우리 "#01, #02" 구조가 이것
  - 제목 짓기: 항상 "주제 + 시사점" ('그래서 뭐?'가 제목에 있음)
  - 인포그래픽 시리즈를 본 보고서와 병행 → 나중 카드뉴스의 근거

## 2. Our World in Data — 데이터 퍼블리싱 구조 철학
- 페이지 설계 원칙: overview first → zoom/filter → details-on-demand (핵심 인사이트 맨 위, 방법론은 아래로)
- 모든 차트에 데이터 출처 명시. 원본 가공(집계·변환) 시 그 사실도 표기.
- 훔칠 것:
  - "핵심 요약 맨 위 + 방법론 부록" 구조 (이미 #01에 적용함)
  - 차트마다 출처 한 줄
  - 보고서 쌓이면 "주제 허브"(관련 보고서 모음) 구조
